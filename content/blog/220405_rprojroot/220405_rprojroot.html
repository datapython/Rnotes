--- title: "rprojroot makes setwd() for Rmarkdown project so easy" author: "Shengguo" date: "2022-04-07 19:52:08" categories: ["R"] tags: ["R Markdown", "rprojroot"] ---

Recently, just found that the package rprojroot is very useful when working on a Rmarkdown file. See detail. It makes setwd() for Rmarkdown project so much easier by simply adding this line of code into your first code chunk.

knitr::opts_knit$set(root.dir = rprojroot::find_rstudio_root_file())

Inside a Rmarkdown project

# To find the root directory for the root the project when using RStudio.
rprojroot::find_rstudio_root_file()  
## [1] "/home/sl1016/SLfiles/all_R/myproj/blogs/Rnotes"

# To check if a sub-directory exists
rprojroot::has_dir("data")
## Root criterion: contains a directory "data"

# To check if a file exists
rprojroot::has_file("index.md")
## Root criterion: contains a file "index.md"

Similarly, it works for package project, rprojroot::find_package_root_file().
